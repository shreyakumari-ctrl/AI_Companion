# AI Companion 🤖

AI Companion is a lovable AI chatbot designed to help users with real-life tasks and friendly conversations.

## 🌟 Features
- 💬 Friendly chat interface
- 😊 Emotion-based responses (basic)
- 📱 Responsive design (mobile-friendly)
- ⚡ Fast and simple UI

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (handled by team)
